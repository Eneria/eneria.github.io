CSS Color Properties Reference

Man, colors are fun. Here, I've used value shifting to keep the text color varied, but still analogous. What's nice is CSS has plenty of options to add those touches to various elements. I quite enjoy using gradients because seeing the colors gently shift into another color makes my silly little monkey brain neurons activate.

Below I have compiled a table of the various properties that can be used in CSS to play with those funky wavelengths of light. This table will not be comprehensive, just a representation of the knowledge I have accumulated and found useful so far.

Properties

Property Name Description
accent-color: Accent Color Specifies an accent color for user-interface controls
background-color: Background Color Specifies the background color of an element
border-color: Border Color Sets the color of all four border sides. *Note, this property can be used to specify certain border sides--i.e "border-top-color:"
border-inline-color: Border Inline Color Sets the color of the borders at start and end in the inline direction. *Note, this can be specified for the start or end of the inline.
color: Color Sets the color of text.
column-rule-color: Column Rule Color Specifies the rule of the color between columns
outline-color: Outline Color Specifies the color of an outline
text-decoration-color Text Decoration Color Specifies the color of the text decoration.

I find that colors are one of the most essential points of learning if you plan on doing anything with design. A small accent of color can make the difference between making something pop, or just falling into another wall of text. You could say it provides another layer of emphasis.
